Sentiment Toward Local Courts: A Demographic Analysis

Project Information

  • Category: Data Wrangling / Visualization
  • GitHub URL: github.com/PatelShivam26684/DSProj1

Project Overview:

This project leverages the General Social Survey (GSS) dataset to explore how demographics influence perceptions of local court harshness. A comprehensive data cleaning pipeline was used to prepare over 54,000 valid responses for analysis across variables such as sex, race, education, and perceived social class.

Key Steps:

  • Removed non-informative/null responses (e.g., “Don’t know”, “No answer”)
  • Renamed variables for readability (e.g., courts → Courts Deal With Criminals)
  • Filtered and validated categorical variables for analysis

Insights:

  • The vast majority across all groups believed courts are "Not Harsh Enough"
  • Black and lower-class respondents were more likely to say courts are "Too Harsh"
  • Higher education was correlated with more moderate views of court fairness

Tools & Libraries:

  • Python, Jupyter Notebooks
  • Pandas, Matplotlib, Seaborn
