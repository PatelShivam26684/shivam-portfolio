Predicting Stroke Occurrence Using Machine Learning

Project Information

  • Category: Predictive Modeling / ML
  • GitHub URL: github.com/PatelShivam26684/DSProj2

Project Overview:

This project built and evaluated machine learning models to predict stroke occurrence using a structured healthcare dataset. The primary goal was to compare model performance using R² and RMSE, and determine the most effective model type.

Data Wrangling & Cleaning:

  • Dropped null BMI values; converted bi-categorical variables to binary (0/1)
  • Applied winsorization to cap outliers in glucose and BMI variables
  • Normalized inputs for KNN model

Models Tested:

  • Linear Regression — Best performing model (R² = 0.0702, RMSE = 0.1861)
  • Classification Tree — Poor performance (R² = -0.0403 at best)
  • K-Nearest Neighbors — R² = -0.0129, RMSE = 0.1942 (best at k = 9)

Key Findings:

  • No model outperformed baseline values (R² = 0.0872, RMSE = 0.2060)
  • Linear regression consistently outperformed tree and KNN models
  • Most predictive features: Age, Hypertension, Heart Disease, Glucose Level

Tools & Libraries:

  • Python, Jupyter Notebooks
  • Pandas, Sklearn, Matplotlib
