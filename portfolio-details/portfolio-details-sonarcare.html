Project Overview:

SonarCare is a full-stack AI-powered remote patient monitoring (RPM) app designed to assist patients, nurses, and care teams. It enables real-time clinical triage, structured profile generation from notes, patient-specific education, and care coordination through a modern dashboard.

Key Features:

  • Personalized AI assistant powered by Perplexity’s SONAR API
  • Chat interface with SonarCare for triage and condition-specific questions
  • Patient dashboards for vitals, boredom education, and emergency alerts
  • Care team dashboard for patient monitoring and AI-guided summaries

Technical Highlights:

  • React (frontend), Flask (backend), PostgreSQL (Render-hosted DB)
  • Role-based routing for patients, nurses, and admins
  • Integrated medical grade prompts with Perplexity’s `sonar-reasoning` model
  • Custom profile parser and alert system using clinical thresholds (LOINC, ADA, AHA, etc.)
